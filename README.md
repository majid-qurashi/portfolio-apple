# 💻 Portfolio Website

A modern and interactive portfolio website with a macOS-inspired interface.

## 🚀 Features

- Built using Astro, React, and Tailwind CSS  
- macOS-style terminal interface with chatbot  
- Easy customization using `userconfig.ts`  
- Projects section with screenshots and GitHub links  
- Notes section for storytelling content  
- Fully responsive design  
- SEO optimized  
- TypeScript support 
- Ready for Vercel deployment  

## 🛠️ Tech Stack

- Astro  
- React  
- Tailwind CSS  
- TypeScript  
- Vercel  

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio
cd portfolio
```
